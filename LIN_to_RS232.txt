LIN_to_RS232 ver. 1.0 2022-02-21

Aviable baudrate value for lin: 9600, 19200
RS232 baudrate: 115200

PinOut:

USART0 (USART_PC)
	PA9 - TX
	PA10 - RX
USART1 (USART_LIN)
	PA2 - TX
	PA3 - RX

Доступные команды конфигурации:
1) 'b' 96 - set baud 9600
2) 'b' 192 - set baud 19200
3) 'e' 0 - classic CRC 
4) 'e' 1 - enhanced CRC
5) 'c' 0 - disable show invalid CRC
6) 'c' 1 - enable show invalid CRC
7) 'i' 0 - get current info

Доступные режимы отправки пакетов:
1) s 0 PID + DATA - отправка frame slave без ожидания совпадения PID (не соответствует протоколу!)
2) s 1 PID + DATA - отправка frame slave с ожиданием совпадения PID (приходящие пакеты пробрасываются в RS232)
3) m PID + DATA - отправка master packet 

Все команды не зависят от регистра, после конфигурации сохраняются на последнюю страницу main flash (0x0801FC00) в порядке 
CRC + filt + baud по 32 бита на каждую переменную. 

При приеме, в случае установки 'c' 0 пакеты с некорректным CRC просто не выводятся. В случае, если установлен режим 'c' 1, то в случае приема
некорректного пакета, он будет выведен в виде 0x00 PID DATA CRC
